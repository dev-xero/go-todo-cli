# `GO TODO CLI`
A light-weight and fast in-memory todo-list application, written in GO.

## Usage

```bash
AVAILABLE COMMANDS:

todo > new [something to add]
todo > list
todo > check [index]
todo > remove [index]
```

## 1. Add an item: new
add an item to the todo-list, syntax:
```bash
todo > new [item]
todo > :n [item]
```
